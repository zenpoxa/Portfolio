Manipulation de données

Gérer des données de l'information

Késako ?

En duo, nous avons conçu, codé, peuplé puis analysé une base de données relative à des établissements scolaires, afin de réaliser des statistiques fondamentales, en mesurant les niveaux de corrélation.

Objectif : Comprendre le poids des grands jeux de données, et prendre le temps d'analyser.

Étapes

1. Concevoir la base de données

2. Créer et peupler la base

3. Extraire une portion de données

4. Faire des analyses statistiques en python

5. Rédiger un rapport en LaTeX

Acquis

J'ai affiné ma compétence de conception, car c'est la fondation même d'une base de données.

Nous avons évidemment utlisé plusieurs notions mathématiques pour mener à bien les analyses statistiques : covariance, corrélation, régression linéaire multiple.

Bien évidemment, mon niveau en SQL, en python et en LaTeX n'a pu qu'être amélioré.
